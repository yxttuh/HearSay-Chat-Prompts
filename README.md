# **HearSay Chat Prompts**
### A Jackbox Party Pack 11 HearSay custom prompt injector that takes prompts from your Twitch chat.

## **Features**
- Add any Twitch chat via text box
- Remove any prompt you desire with the "remove" button
- Only accepts prompts from roles: Sub, VIP, Mod, Broadcaster

## **Usage**
- Download and run the latest HearSay-Chat-Prompts.exe in [Releases](https://github.com/yxttuh//HearSay-Chat-Prompts/releases)
- Add your Jackbox Party Pack 11 folder via the selector
- Add your Twitch channel via the text box
- Type !prompt (your prompt here) to add it to the game
